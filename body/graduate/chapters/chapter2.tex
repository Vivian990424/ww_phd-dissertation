\chapter{数据源与预处理}

本章系统介绍本研究使用的全部数据来源、基本特征及预处理流程，为后续方法构建与结果分析奠定数据基础。相较于期刊论文中压缩的数据描述，博士论文需要将数据体系阐述得更加完整和清晰。全部数据预处理过程基于Python编程环境（3.8及以上版本）实现，核心处理库包括GDAL、Rasterio、NumPy和SciPy，并通过自动化批处理脚本实现多源数据的高效协同处理。

\section{全球水稻种植制度与物候差异}

\subsection{全球主要水稻种植区概述}

全球水稻是世界上最重要的粮食作物之一，养育了全球一半以上的人口。据联合国粮农组织（FAO）统计，全球水稻种植面积长期稳定在1.6亿公顷以上，年产量超过7亿吨 \cite{GRiSP2013}。从农业生态背景来看，FAO农业生态区（Agro-Ecological Zones, AEZ）框架按照温度带（热带、亚热带、温带）与水分状况（干旱、半干旱、亚湿润、湿润）对全球农地进行系统划分；水稻作为典型的高耗水、高温需求作物，主要分布于湿润（humid）与亚湿润（sub-humid）生态区，尤以热带与亚热带地区的低地最为集中 \cite{FAOAEZ}。在空间生产格局上，全球水稻呈现极度不均衡的集聚特征：亚洲贡献了全球约90\%的产量，其中中国、印度、印度尼西亚、孟加拉国、越南、泰国和缅甸七国的种植面积与产量合计占全球的80\%以上；非洲（尤其是西非和东非）与南美洲（以巴西为代表）是近二十年来水稻面积扩张最显著的区域；北美洲（美国）、欧洲（意大利、西班牙等）及大洋洲（澳大利亚）的种植规模相对有限，但代表了高度机械化、高投入产出的现代农业模式 \cite{GRiSP2013,Maclean2002}。

从农业生态系统类型来看，国际水稻研究所（IRRI）与全球水稻科学伙伴关系（GRiSP）依据水分来源与地形条件，将全球水稻划分为四大生态系统 \cite{GRiSP2013,Maclean2002}：
\begin{enumerate}
    \item \textbf{灌溉低地稻（irrigated lowland）}：约占全球水稻种植面积的55\%，是最主要的生态系统类型。该类稻田主要分布于大型河流三角洲与冲积平原，具备完善的灌溉基础设施，如中国长江中下游平原、印度恒河平原、东南亚湄公河与伊洛瓦底江三角洲等。稳定的水分供应使其物候周期相对规律，种植制度以一年一熟或双季稻为主。
    \item \textbf{雨养低地稻（rainfed lowland）}：约占全球面积的25\%，广泛分布于东南亚与南亚的季风区，如泰国东北部、柬埔寨、老挝及印度东部沿海。其生产高度依赖季风降雨，产量受降水时空分布的显著制约，物候启动时间与雨季来临密切相关，年际波动较大。
    \item \textbf{雨养旱地稻/山地稻（rainfed upland）}：约占12\%，主要分布于丘陵坡地，如东南亚北部山区、南美巴西高原及非洲部分区域。该类稻田缺乏持续淹水层，土壤水分以降雨和地下水为主，光谱特征与旱作谷物更为接近，遥感识别难度显著增加。
    \item \textbf{深水稻与浮稻（deep water and floating rice）}：历史上约占8\%，但近年来因水利设施改善与种植结构调整而持续减少，目前主要残存于孟加拉国、缅甸伊洛瓦底江下游及湄公河三角洲的洪泛区。其生长季伴随水位上涨，植株可随水面升高而快速伸长，物候模式与普通低地稻差异显著。
\end{enumerate}

从洲际尺度审视，亚洲是全球水稻生产的绝对核心。东亚（中国、日本、韩国）以灌溉低地稻为主，种植制度从东北温带单季稻到华南热带双季稻乃至三季稻，复种指数差异悬殊；南亚（印度、孟加拉国、巴基斯坦）与东南亚（泰国、越南、印尼、缅甸、菲律宾）则同时囊括了灌溉低地、雨养低地与深水稻等多种类型，是全球水稻生态系统多样性最高的区域。非洲稻区可分为西非（尼日利亚、几内亚、塞拉利昂等）和东非（马达加斯加、坦桑尼亚等）两大片，以雨养低地和雨养旱地稻为主，种植制度多样且田间地块破碎化严重，区域数据基础与系统研究相对薄弱。美洲稻区中，美国密西西比三角洲与加利福尼亚中央谷地代表了大农场、高度机械化的灌溉水稻生产；巴西、乌拉圭及阿根廷等南美国家则在水热条件适宜的平原区持续扩张水稻面积。欧洲稻区集中于地中海沿岸，如意大利波河平原、西班牙瓦伦西亚及埃布罗河流域、希腊塞萨洛尼基平原等，面积虽小，但集约化程度高。大洋洲以澳大利亚新南威尔士州和昆士兰州的灌溉稻为主，采用大规模旱作直播与飞机施肥等现代化管理手段 \cite{GRiSP2013,Maclean2002}。

上述全球水稻种植格局的空间分异，对遥感识别构成了根本性的挑战。首先，四大生态系统在光谱特征与物候规律上存在本质差异：灌溉低地稻在移栽期具有显著的冠层淹水信号，而雨养旱地稻则缺乏这一关键特征；深水稻的物候时间窗口与洪水节律耦合，与普通低地稻截然不同。其次，亚洲季风区复杂的复种制度（单季稻、双季稻、三季稻及再生稻）导致同一像元内可能出现多期水稻生长信号的叠加或交错，难以用固定的时间窗口或阈值统一刻画。再次，非洲等新兴稻区云雨覆盖频繁、田间地块破碎、农业统计数据匮乏，不仅制约了光学遥感的可用性，也缺乏足够的地面验证样本支撑高精度制图。因此，构建具有全球普适性的水稻遥感识别方法，必须充分考虑生态类型差异、物候空间异质性及区域数据可获得性，这也是本研究在方法设计中所要回应的核心问题。

\subsection{水稻物候的区域差异性}

不同地区水稻的播栽期、生长季长度和复种制度存在显著差异。例如，温带地区（如中国东北、日本北部）通常为单季稻，生长季集中在夏季，物候周期相对统一；亚热带地区（如中国长江流域、印度东部）可实现双季稻（早稻和晚稻），一年内存在两个明显的水稻生长峰期；热带地区（如泰国、越南湄公河三角洲）则可能实现三季稻或多年生稻，生长季全年分布，物候信号交错叠加。这种物候差异是全球水稻制图面临的首要挑战，也是本研究不能简单使用固定时间窗识别水稻的根本原因。准确刻画全球稻区物候的时空异质性，是构建具有全球普适性遥感识别方法的前提条件。

\section{遥感数据}

\subsection{植被指数数据来源与基本特征}

本研究用于构建归一化植被指数（NDVI）和陆地表面水分指数（LSWI）时间序列的主要遥感数据源为MODIS地表反射率产品（MOD09GA / MYD09GA）。MODIS植被指数因其良好的辐射定标精度和全球覆盖能力，已成为大尺度植被监测的标准数据源 \cite{Huete2002}。该数据的空间分辨率为500 m，时间分辨率为1天，时间范围覆盖2000—2020年。LSWI基于短波红外波段构建，对植被冠层和土壤液态水含量敏感，能够有效捕捉稻田移栽期的淹水信号 \cite{Gao1996}。NDVI和LSWI的计算公式如下：
\begin{equation}
    \text{NDVI} = \frac{\rho_{\text{NIR}} - \rho_{\text{red}}}{\rho_{\text{NIR}} + \rho_{\text{red}}}
\end{equation}
\begin{equation}
    \text{LSWI} = \frac{\rho_{\text{NIR}} - \rho_{\text{SWIR1}}}{\rho_{\text{NIR}} + \rho_{\text{SWIR1}}}
\end{equation}
其中，$\rho_{\text{red}}$、$\rho_{\text{NIR}}$ 和 $\rho_{\text{SWIR1}}$ 分别对应红光、近红外和短波红外波段的地表反射率。

\subsection{质量控制与云噪声处理}

MODIS地表反射率产品包含详细的质量控制（QC）标识，用于标识每个像元观测状态的可靠性。本节基于QC信息剔除云、云阴影、气溶胶高污染等低质量观测，仅保留高质量的晴空（clear-sky）像素用于后续分析。具体而言，利用MOD09GA/MYD09GA产品自带的1 km分辨率QC层，筛选出"完全确定"和" probably确定"的云掩膜状态，并进一步剔除内部云掩膜算法标记为云及云阴影的像元。对于气溶胶污染，优先保留低气溶胶负荷条件下的观测，在数据稀缺的稻区适当放宽至中等气溶胶条件，以平衡数据质量与时空覆盖度。

\subsection{日尺度时间序列重建}

为获得时空连续的日尺度植被指数时间序列，本研究采用以下重建方法：
\begin{itemize}
    \item \textbf{NDVI时间序列：} 采用Savitzky-Golay滤波平滑重建，保留植被生长的季节性特征同时抑制噪声；
    \item \textbf{LSWI时间序列：} 采用线性插值填补缺失值，保持淹水信号的连续性。
\end{itemize}

时间序列重建的完整流程包括：原始数据读取 → 质量控制筛选 → 指数计算 → 时序滤波/插值 → 输出日尺度时间序列。在NDVI时序重建中，Savitzky-Golay滤波以其能够在平滑噪声的同时保留局部峰值特征的优势被选用，其窗口大小和多项式阶数通过交叉验证确定。对于LSWI序列，考虑到淹水信号的持续性和突变性，线性插值能够在保持信号趋势的同时有效填补数据空白。本节亦对比了Whittaker平滑和双逻辑斯蒂拟合等方法的适用性，最终基于对水稻物候信号保真度和计算效率的综合考量，确定了上述重建方案。

\section{辅助数据}

\subsection{耕地掩膜与土地覆盖数据}

为约束水稻搜索范围、减少计算量并降低误判，本研究采用多种土地覆盖产品构建耕地掩膜：
\begin{itemize}
    \item MCD12Q1（MODIS土地覆盖产品，500 m，年尺度）；
    \item GLC-FCS30（全球30 m精细地表覆盖产品）；
    \item 其他辅助耕地图层（如FAO耕地统计格网化数据）。
\end{itemize}

不同产品的使用方式如下：首先利用MCD12Q1的IGBP和UMD分类方案提取耕地类别，构建全球尺度的基础耕地掩膜；随后引入GLC-FCS30的高分辨率耕地信息进行空间降尺度验证，在重点区域校正MCD12Q1的分类误差。通过交叉验证排除明显非耕地区域（如水体、城市、永久冰雪等），对于不同产品之间的不一致性，以MCD12Q1为主参考，在争议区域结合DEM坡度和已有水稻分布先验知识进行仲裁判定，最终生成覆盖全球稻区的统一耕地潜在区掩膜。

\subsection{国家统计数据}

联合国粮农组织（FAO）国家尺度水稻种植面积和产量统计数据用于宏观层面的校准与验证。FAO数据覆盖全球主要水稻生产国，时间跨度为2000—2020年，空间粒度为国家行政单元。在校准环节，将遥感制图结果按国界聚合为国家级水稻面积统计值，与FAO官方统计数据进行线性回归和相关性分析，评估全球制图结果在宏观尺度上的可靠性。对于FAO统计中存在的年份缺失或异常值，采用相邻年份线性插值进行填补，以确保长时间序列分析的连续性。

\subsection{已有高分辨率水稻参考图}

收集已有的高分辨率水稻空间分布图作为验证样本的主要来源，包括基于Landsat或Sentinel-2数据研制的水稻图（如NESEA-Rice10 \cite{Han2022}、中国/印度长时序水稻图 \cite{Zhang2015} 等）。参考图的空间覆盖范围聚焦亚洲主要稻区，制图年份以2015—2020年为主，总体精度普遍高于85\%。验证样本的提取遵循空间独立性和类别平衡性原则，即在参考图覆盖范围内按分层随机策略抽取验证点，确保水稻与非水稻样本比例合理，并避免验证点落在训练样本的空间邻域内。

\subsection{DEM与地形数据}

数字高程模型（DEM）数据采用SRTM（Shuttle Radar Topography Mission）V3版本，空间分辨率为30 m，覆盖全球陆地区域（60°N至56°S）。在本研究中，DEM主要用于辅助分析水稻分布的地形约束条件，如平原稻田与山地梯田的海拔和坡度差异，以及在后文中分析地形对温度效应的空间异质性影响。SRTM 30 m数据通过重采样与聚合生成500 m分辨率的地形辅助图层，与GlobalRice500数据集保持空间一致。

\section{LST与地形数据}

\subsection{地表温度数据}

本研究使用的地表温度（LST）数据来源于MODIS地表温度产品（MOD11A1 / MYD11A1），具体参数如下：
\begin{itemize}
    \item 数据源：MODIS地表温度产品（MOD11A1 / MYD11A1）；
    \item 时间范围：2003—2020年；
    \item 时间分辨率：日尺度；
    \item 空间分辨率：1 km；
    \item 包含昼间和夜间两个时相的LST观测。
\end{itemize}

LST数据的质量控制流程包括：利用QC层剔除受云污染和质量不合格的像元，保留"良好质量"和"其他质量"等级的观测。对于云污染严重的时段，采用时间维度的线性插值进行填补。昼间和夜间LST在能量平衡分析中具有不同的物理意义：昼间LST受太阳辐射和地表能量分配的直接影响更为显著，对蒸散发变化响应敏感；夜间LST则更能反映土壤热惯性和水分热容量的综合效应，两者结合可全面刻画稻田对地表温度的日变化调节作用。

\subsection{数据重采样与空间匹配}

由于GlobalRice500数据集的空间分辨率为500 m，而原始LST数据为1 km，需要将LST和DEM数据重采样至500 m的MODIS tile格式。LST数据采用双线性插值方法进行降尺度，该方法能够在保留温度空间渐变特征的同时，减少最近邻法引入的块效应。DEM数据则通过最近邻法重采样，以保持地形边界的空间准确性。在重采样过程中，严格统一所有数据的空间参考（WGS 84地理坐标系）和投影系统（等面积或等经纬度），确保多源数据在像素级别上的严格空间匹配，为后续基于像元的稻田温度效应分析奠定数据基础。

本节以完整流程图的形式呈现全部数据预处理环节，各环节之间的逻辑关系与数据流转路径如图~\ref{fig:data_flow} 所示。

    \centering
    \includegraphics[width=0.95\linewidth]{figure/chapter2/data_flow.pdf}
    \caption{数据预处理总体流程}
    \label{fig:data_flow}

流程图包含以下关键环节：
\begin{enumerate}
    \item 原始遥感数据获取（MOD09GA / MYD09GA / MOD11A1 / MYD11A1）；
    \item 质量控制（QC）筛选与低质量观测剔除；
    \item NDVI / LSWI 计算与指数构建；
    \item 日尺度时间序列重建（Savitzky-Golay滤波 + 线性插值）；
    \item 耕地掩膜构建与水稻潜在区提取；
    \item 参考水稻图处理与验证样本生成；
    \item FAO统计数据整理与空间聚合；
    \item LST与DEM重采样及空间匹配；
    \item 输出统一格式、统一分辨率的标准化数据集。
\end{enumerate}

此外，全部数据预处理过程在Linux服务器集群上基于Python / GDAL自动化处理框架完成。原始HDF格式MODIS数据通过GDAL Python绑定读取并提取目标波段，质量控制与指数计算环节采用NumPy向量化运算加速，时序重建与重采样流程通过多进程并行处理提升效率。预处理脚本按数据源模块化组织，各模块输出遵循统一的NetCDF或GeoTIFF格式规范，便于后续方法章节的直接调用与分析。
