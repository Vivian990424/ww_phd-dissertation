\chapter{全球水稻日尺度遥感识别方法：MPD\_DTW 框架}

本章是本研究的方法创新核心，详细阐述融合移动物候检测（Moving Phenological Detection, MPD）与动态时间弯曲（Dynamic Time Warping, DTW）的全球水稻日尺度遥感识别方法——MPD\_DTW 框架。近二十年来，全球水稻遥感制图方法经历了从单一阈值法到复杂时间序列分析的演变\cite{Dong2016}，但如何在统一框架内兼顾物候自适应检测与柔性曲线匹配仍是亟待解决的关键科学问题。本研究提出的MPD\_DTW框架正是针对这一问题的系统性解决方案。

\section{全球水稻制图的主要挑战}

在全球尺度开展水稻遥感制图面临三大核心挑战，这些挑战直接决定了方法设计的基本逻辑。已有研究表明，全球水稻种植的高度时空异质性是制约大尺度制图精度的首要瓶颈\cite{Dong2016,Xiao2006}，而现有方法在应对这种异质性方面存在各自的局限性。

\subsection{水稻种植时间不固定}

全球不同稻区的水稻播栽期差异显著，从热带地区的全年多季种植到温带地区的单季种植，不同区域的实际播栽期可相差数月之久。即使在同一地区，受气候年际波动、灌溉条件变化、品种更替和管理策略调整的影响，水稻的实际播栽时间也存在较大的年际和区域内变异\cite{Xiao2006}。传统基于固定时间窗的遥感识别方法难以捕捉这种动态变化，极易在非水稻生长季产生错分，或在实际水稻季产生漏检。因此，发展能够自适应追踪种植时间变异的方法成为全球水稻制图的首要需求。

\subsection{水稻生长季长度差异大}

从热带地区90天左右的短季稻到温带地区150天以上的长季稻，全球水稻生长季长度跨度极大\cite{Massey2017}。复种制度的存在进一步加剧了这种差异：双季稻和三季稻的每一季稻生长季都比单季稻更短。基于固定长度曲线匹配的方法在全球推广时面临根本性困境：统一截取短窗口会丢失长季稻的关键生育期信息，而采用长窗口则会在短季稻序列中引入大量无效填充值\cite{Onojeghuo2018}。这种时间尺度不匹配问题直接限制了传统时间序列分析方法在全球稻区的普适性。

\subsection{单一方法的局限性}

\begin{itemize}
    \item \textbf{单一物候阈值方法} 虽具有可解释性强、计算高效的优点，但阈值设定对区域和年份敏感，难以同时适应全球多样的种植制度\cite{Dong2016,Xiao2006}；
    \item \textbf{单一曲线匹配方法} 虽能利用完整生长季信息，但固定长度的曲线比较无法处理生长季长度差异，且对曲线起始点的敏感性高\cite{Petitjean2012}。
\end{itemize}

因此，本研究提出将物候检测与曲线匹配有机结合的思路：先通过物候规则自适应地确定每个像元每个潜在生长季的SOS和EOS，再通过DTW对不同长度的曲线进行柔性匹配，从而实现兼顾普适性与精度的全球水稻制图。

\section{Moving Phenological Detection 模块}

MPD模块是MPD\_DTW框架的第一阶段，其核心任务是逐像元、逐日检测水稻生长季的起始（SOS）和结束（EOS）时间。该模块的设计灵感源于经典的水稻物候检测方法\cite{Xiao2006}，但通过逐日移动搜索策略实现了对种植时间高度变异区域的自适应检测。

\subsection{SOS 检测原理}

水稻的起始季（SOS）主要基于移栽期或淹水信号进行识别。在移栽期，稻田经历从裸水到植被快速建立的过程，表现为LSWI迅速上升且LSWI与NDVI出现特定的数值关系\cite{Xiao2006,Gao1996}。MODIS植被指数产品经过严格的辐射定标和大气校正，为长时序物候分析提供了稳定的数据基础\cite{Huete2002}。具体判定依据包括：
\begin{itemize}
    \item LSWI显著上升，反映稻田淹水状态的确立；
    \item LSWI与NDVI的相对大小关系（如LSWI $\geq$ NDVI或LSWI/NDVI比值超过阈值），反映水体信号强于植被信号的特征；
    \item 可结合"洪水因子"（F）进一步刻画淹水信号的强度。
\end{itemize}

\subsection{EOS 检测原理}

水稻的结束季（EOS）主要根据NDVI的局部下降和收获后生物量降低来识别。与起始季不同，成熟收获期的光谱特征主要表现为植被指数的系统性下降\cite{Massey2017}。具体判定依据包括：
\begin{itemize}
    \item NDVI从峰值显著下降，反映作物成熟和收获；
    \item NDVI下降后的稳定低值状态，反映收获后的裸土或休闲期；
    \item 结合生长季长度约束（T1、T2），排除不合理的短周期噪声。
\end{itemize}

\subsection{逐日搜索策略}

MPD模块采用逐日滑动的搜索策略：在每个像元的时间序列上，从时间序列的起始日期开始逐日判断SOS条件是否满足；一旦SOS被确认，随即在该SOS之后搜索满足EOS条件的日期。这一过程在单个像元上循环执行，直至覆盖整个时间序列。这种逐日搜索机制是"移动检测"思想的基础，确保算法能够自适应地捕获不同播栽期的水稻季，而无需预先设定固定的搜索窗口。

\section{Dynamic Time Warping 模块}

DTW模块是MPD\_DTW框架的第二阶段，其核心任务是在MPD确定SOS--EOS后，对该时段的NDVI曲线进行水稻身份的验证。DTW作为一种经典的时间序列相似性度量方法，在遥感地物分类中已展现出显著优势\cite{Petitjean2012,Maus2016,Belgiu2018}。

\subsection{为什么需要 DTW}

在全球不同地区，水稻生长季长度差异显著。直接比较固定长度的NDVI曲线（如欧氏距离或简单相关系数）存在明显缺陷：
\begin{itemize}
    \item 若截断至最短长度，会丢失长季稻的关键生育期信息；
    \item 若统一至最长长度，短季稻曲线会产生大量无意义的填充值；
    \item 不同地区水稻各生育阶段（分蘖、拔节、抽穗、灌浆）的持续时间不同，固定时间轴的比较难以对齐关键特征点。
\end{itemize}

动态时间弯曲（DTW）能够对两条不同长度的时间序列进行非线性对齐，通过弯曲时间轴使得曲线的相似性评估更加灵活和准确\cite{Petitjean2012}。这一特性使DTW特别适合处理全球稻区生长季长度的巨大变异。

\subsection{DTW 距离计算}

在MPD确定某一潜在生长季的SOS和EOS后，提取该时段的NDVI子序列，将其与预先构建的标准水稻NDVI参考曲线进行DTW匹配。DTW距离的计算包括：
\begin{itemize}
    \item 构建累积距离矩阵，描述序列点之间的局部相似性；
    \item 采用动态规划寻找使整体累积距离最小的最优弯曲路径\cite{Petitjean2012}；
    \item 输出DTW距离作为曲线相似度的度量。
\end{itemize}

近年来，研究者进一步发展了时间加权DTW（TWDTW）和导数DTW（DDTW）等改进算法以提升物候匹配精度\cite{Maus2016,Ye2023}。本研究采用标准DTW算法，因其对全球不同稻区生长季长度变异具有最佳的适应性和计算效率。标准水稻参考曲线基于已有高分辨率水稻图和地面观测代表性样点构建\cite{Han2022,Zhang2015}，以确保模板的区域代表性和物候真实性。

\subsection{水稻判定准则}

设定DTW距离阈值，将DTW距离小于阈值的SOS--EOS时段判定为水稻，反之则判定为非水稻。阈值的选择直接决定了制图结果的严格程度，将在第\ref{sec:sensitivity}节中通过ROC曲线分析和精度权衡进行详细讨论。

\section{MPD 与 DTW 的反馈机制}

\label{sec:feedback}

MPD与DTW之间的反馈机制是本研究方法最重要的逻辑创新，也是博士论文中需要比期刊论文阐述得更加深入的部分。该机制通过DTW验证结果动态调控MPD的搜索策略，实现了物候检测与曲线匹配的闭环协同。

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.9\textwidth]{figure/chapter3/mpd_dtw_feedback.pdf}
    \caption{MPD\_DTW反馈机制完整流程图，包含正向反馈与负向反馈两种策略。}
    \label{fig:mpd_dtw_feedback}
\end{figure}

\subsection{正向反馈：水稻确认后的移动策略}

若当前SOS--EOS时段的NDVI曲线经DTW判定为水稻，则下一次SOS搜索从当前EOS之后开始。这一正向反馈机制确保：
\begin{itemize}
    \item 已识别的水稻季不会被重复检测；
    \item 不同季节之间不会出现重叠，适应复种稻区的多时相检测需求；
    \item 搜索过程沿时间轴"移动"前进，体现"Moving"的核心思想。
\end{itemize}

\subsection{负向反馈：非水稻排除后的移动策略}

若当前SOS--EOS时段的NDVI曲线经DTW判定为非水稻，则下一次SOS搜索从当前SOS后一天继续开始。这一负向反馈机制确保：
\begin{itemize}
    \item 不因为一次非水稻判定而跳过潜在的真实水稻季；
    \item 时间轴继续精细滑动，避免漏检；
    \item 系统对误检具有容错和恢复能力。
\end{itemize}

\subsection{反馈机制的完整流程}

综合上述两种反馈策略，MPD\_DTW的完整工作流程如下：
\begin{enumerate}
    \item 初始化搜索起点 $t = t_{\text{start}}$；
    \item MPD在$t$之后搜索SOS，若找到则继续搜索EOS，形成候选生长季$[\text{SOS}, \text{EOS}]$；
    \item DTW将该生长季的NDVI曲线与标准水稻曲线匹配，计算DTW距离；
    \item 若DTW距离$<$阈值，判定为水稻，更新$t = \text{EOS} + 1$；否则判定为非水稻，更新$t = \text{SOS} + 1$；
    \item 重复步骤2--4，直至覆盖整个时间序列；
    \item 输出该像元所有被判定为水稻的SOS、EOS及对应的DTW距离。
\end{enumerate}

这一反馈机制使得MPD\_DTW能够自适应地处理单季稻、双季稻和多季稻，无需预先指定种植制度信息。

\subsection{洪水因子 F 的确定}

F是SOS检测中用于刻画淹水信号强度的关键参数，其物理本质反映了移栽期稻田水体信号相对于植被信号的占优程度\cite{Xiao2006,Gao1996}。本节说明：
\begin{itemize}
    \item F的物理含义与取值范围：基于典型水稻样点的LSWI--NDVI散点分析确定F的物理阈值；
    \item 不同F取值对SOS检测率和误判率的影响：F值过低会引入大量非水稻淹水信号的伪检测，而F值过高则可能导致弱淹水信号稻区的漏检；
    \item 基于全球典型稻区的交叉验证，确定F的最终取值及其区域调整策略。
\end{itemize}

\subsection{生长季长度约束 T1、T2 的本地化}

T1和T2分别定义了水稻生长季长度的下限和上限，是排除噪声干扰的重要约束条件\cite{Massey2017,Onojeghuo2018}。本节说明：
\begin{itemize}
    \item 全球不同稻区生长季长度的统计特征：基于已有文献和参考数据的统计分析，热带短季稻与温带的生长季差异显著；
    \item 如何基于已有文献和参考数据为不同区域设定本地化的T1、T2；
    \item T1、T2过宽或过窄对检测结果的影响：过宽引入噪声，过窄漏检特殊品种。
\end{itemize}

\subsection{DTW 阈值的选择}

DTW阈值直接决定水稻判定的严格程度，是连接算法输出与制图结果的关键决策参数\cite{Maus2016,Belgiu2018}。本节说明：
\begin{itemize}
    \item 基于训练样本的DTW距离分布确定阈值的方法（如ROC曲线分析）；
    \item 不同阈值下制图精度（用户精度、生产者精度、总体精度）和面积估计的权衡：阈值偏严格有助于提高用户精度但可能降低生产者精度；
    \item 最终阈值的选择依据及其在全球尺度的一致性验证。
\end{itemize}

\subsection{参数敏感性综合分析}

通过控制变量实验，系统展示不同参数组合对以下指标的影响：
\begin{itemize}
    \item 全球及区域水稻面积估计；
    \item SOS和EOS的时空分布特征；
    \item 与验证样本对比的分类精度指标。
\end{itemize}

图\ref{fig:sensitivity_analysis}以图表形式呈现敏感性分析结果，论证所选参数组合的合理性和稳健性。分析结果表明，所选参数组合在保持全球面积估计稳健性的同时，能够在区域尺度实现较高的识别精度。

\section{方法适用性与局限性分析}

\subsection{对不同种植制度的适应性}

讨论MPD\_DTW对以下种植制度的适用性：
\begin{itemize}
    \item \textbf{单季稻：} 搜索过程通常只检测到一个水稻季，方法最直接，物候信号完整清晰；
    \item \textbf{双季稻：} 反馈机制确保两季水稻都能被依次检测，前提是两季之间的休闲期足够明显以产生光谱分界；
    \item \textbf{多季稻/热带稻区：} 一年内可能存在三个或更多水稻季，方法在理论上可扩展，但季间休闲期极短或连续种植时可能出现粘连；
    \item \textbf{温带稻区：} 生长季明确，但受春季低温或秋季早霜影响，物候信号可能偏弱，需结合区域参数调整。
\end{itemize}

\subsection{方法的局限性}

客观讨论MPD\_DTW可能在以下场景中出现不确定性：
\begin{itemize}
    \item \textbf{旱直播稻：} 缺乏移栽期的淹水信号，SOS检测依赖的LSWI--NDVI关系可能不显著\cite{Deng2025}；
    \item \textbf{淹水信号弱的地区：} 如水分管理不佳或雨水稻田，LSWI上升不明显，可能导致SOS漏检；
    \item \textbf{田块破碎地区：} 混合像元效应严重，单一像元内的水稻信号被其他地物稀释，DTW匹配精度下降；
    \item \textbf{云雨频繁地区：} 时序数据缺失严重，即使经过重建，物候信号仍可能被扭曲；
    \item \textbf{与水生植被混淆：} 如芦苇、荷花等湿地植被在光谱和物候特征上与水稻存在一定相似性\cite{Zhan2021}。
\end{itemize}

对上述局限性，可进一步说明本研究在验证环节如何通过多源数据交叉检验来量化和控制这些不确定性。
