# 第二部分：修改说明

原段落已经具备基本机制框架，但整体偏像“结论提要”，四个要点集中在一个长段内，理论层次和章节承接还不够充分。本次修改将其重写为更适合博士论文绪论的连续正文，使读者先理解稻田为什么不是普通旱地农田，再理解为什么不能用全年平均或固定月份概括稻田温度效应，最后理解为什么稻田降温效应需要进入景观结构和对照样本匹配层面的分析。

在科学问题层面，修改后的正文强调稻田是水层、湿润土壤、水稻冠层和人为水分管理共同塑造的水—植被复合下垫面。其地表温度效应不是简单由“水体更冷”导致，而是由水面蒸发、土壤水分维持、冠层蒸腾和显热/潜热分配改变共同决定。这一表述避免将稻田降温机制过度简化，也为后文从能量平衡角度解释昼间 LST 差异提供理论基础。

在章节功能层面，修改后的正文没有提前写入 GlobalRice500 的具体结果或全球平均降温数值，而是只做必要的理论铺垫。对“真实水稻生长季”“昼夜差异”“田块规模”“稻田比例”“邻近对照”等概念的处理，均保持在研究必要性和机制预期层面，用于支撑后续方法章和结果章，而不是提前替代后文结论。

在逻辑衔接层面，修改后的结尾明确说明该部分如何连接下一小节的一般土地系统研究进展，以及如何支撑后续第六章和第七章的生长季约束、样本匹配、昼夜对比和空间结构分析。这样可以使第一章形成更清晰的递进关系：一般地表生物物理机制 -> 稻田特殊性 -> 已有研究进展 -> 研究不足 -> 本文科学问题。

# 第四部分：需要特别避免的表述风险

应避免将稻田地表降温效应表述为能够抵消甲烷排放增温效应。二者属于不同过程、不同尺度和不同指标，除非后续使用统一量纲进行辐射强迫或 CO2 当量核算，否则不能写成直接抵消关系。

应避免把 LST 降温等同于近地气温下降或全球平均气温下降。本文当前正文已经将研究对象限定为稻田相对于周边非稻田农田的地表辐射温度差异，后文仍需保持这一边界。

应避免在绪论中提前写入本文全球平均降温强度、最大降温值、田块规模回归系数或外溢距离等具体结果。绪论中可以提出研究必要性和机制预期，但具体数值应留给第七章结果和第八章讨论。

应避免把稻田降温机制简化为单一“水层降温”。更严谨的表述应同时考虑水面蒸发、土壤水分、冠层蒸腾、热容量、反照率、显热/潜热分配和人为管理制度。

应避免将稻田等同于一般灌溉农田。稻田具有阶段性或长期水层、特有水稻物候和田间水分管理制度，其温度效应应在灌溉研究基础上进一步区分。
